# SEO Checklist

## On-Page SEO Checklist

- [ ] Primary keyword in H1
- [ ] Primary keyword in first 100 words
- [ ] Primary keyword in 2+ H2 headings
- [ ] Keyword density 1-2%
- [ ] 3-5+ internal links included
- [ ] 2-3 external authority links
- [ ] Meta title 50-60 characters
- [ ] Meta description 150-160 characters
- [ ] Article 2000+ words
- [ ] Proper H2/H3 hierarchy (no skipped levels)
- [ ] One H1 per page
- [ ] URL slug contains primary keyword
- [ ] Images have alt text with keywords where natural
- [ ] No broken links

## AI Search Optimization Checklist

- [ ] **Direct answer**: First 1-2 sentences directly answer the target query
- [ ] **Key Takeaways**: TL;DR block with 3-5 specific bullets after intro
- [ ] **Meta description**: Directly answers the query (not just teaser)
- [ ] **YouTube embed**: At least one relevant video suggested
- [ ] **FAQ format**: Questions written in natural language people type into ChatGPT
- [ ] **One idea per section**: Each H2/H3 focuses on single clear concept
- [ ] **Author attribution**: Named author in frontmatter

## Engagement Checklist

- [ ] **Hook**: Opens with question, scenario, stat, or bold statement (NOT generic definition)
- [ ] **APP Formula**: Introduction has Agree, Promise, Preview
- [ ] **Mini-stories**: 2-3 specific scenarios with names, details, outcomes
- [ ] **Contextual CTAs**: 2-3 distributed throughout (not just at end)
- [ ] **First CTA within 500 words**
- [ ] **Paragraph length**: No paragraphs exceed 4 sentences
- [ ] **Sentence rhythm**: Mix of short (5-10 words) and longer (15-25 words)

## Content Quality Checklist

- [ ] No AI filler phrases ("It's worth noting," "In today's landscape," "Delve into")
- [ ] Em-dashes used sparingly (not every paragraph)
- [ ] Contractions used naturally
- [ ] Specific numbers and names (not "many" or "several")
- [ ] Active voice predominant
- [ ] No exclamation point overuse
- [ ] Transitions between sections (not just heading jumps)
- [ ] Conclusion recaps takeaways + CTA

## Technical SEO Checklist (for site audits)

- [ ] HTTPS across entire site
- [ ] Robots.txt allows important pages
- [ ] XML sitemap exists and submitted to Search Console
- [ ] Canonical tags on all pages
- [ ] Core Web Vitals: LCP < 2.5s, INP < 200ms, CLS < 0.1
- [ ] Mobile-responsive design
- [ ] No orphan pages (all pages linked internally)
- [ ] Schema markup (JSON-LD) for key page types
- [ ] Proper heading hierarchy (single H1, logical H2/H3)
- [ ] Image optimization (WebP, lazy loading, compressed)
